# France Football Team Performance Analysis Dashboard (World Cup 2026)

## 📌 Project Overview
This project presents a highly interactive and insightful data analysis dashboard built using **Power BI** to evaluate the squad depth and player performance of the **France Football Team** ahead of the World Cup 2026. The dataset focuses on four key operational areas: Forward Efficiency, Midfield Mastery, Defensive Art, and Goalkeeping Strength.

---

## 📊 Dashboard Pages & Insights

### 1. Forwards Analysis
* **Focus:** Visualizing goal-scoring efficiency, dynamic assist contributions, and penalty metrics.
* **Key Visuals:** Horizontal bar charts comparing goals vs. assists, alongside a dedicated donut chart representing total penalty shares.

### 2. Midfielders Analysis
* **Focus:** Evaluating progressive passes, pass completion rates, and defensive contributions in the middle third.
* **Key Visuals:** Dual-axis bar charts tracking goal contributions and an advanced donut chart mapping won tackles and interceptions per player.

### 3. Defenders Analysis
* **Focus:** Assessing aerial dominance, interceptions, and overall backline structural integrity.
* **Key Visuals:** A premium trendline visual charting **Aerial Won %** and structured cluster bars showing tackle metrics.

### 4. Goalkeeper Analysis
* **Focus:** Tracking clean sheet ratios, shot-saving percentages, and long-ball completion rates.
* **Key Visuals:** Interactive bar segments incorporating advanced **PSxG +/-** metrics to identify elite shot-stopping efficiency under pressure.

---

## 🛠️ Tools & Technologies Used
* **Power BI Desktop:** Dashboard design, data visualization, and UI implementation.
* **Microsoft Excel:** Data storage, restructuring, and source data management.
* **DAX (Data Analysis Expressions):** Customized measures for precise performance metrics.
